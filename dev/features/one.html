Lightning Components: Use Lightning Web Security for LWC & Aura Components (GA)
Contributed by : Subhojeet Ghosh

Lightning Locker has been the default security architecture for Lightning components. LWS started to replace the Lightning Locker for LWC in Sprint '22. Lightning Web Security (LWS) for Aura components is now generally available (GA) too.

However Salesforce will not be automatically enabling LWS for existing orgs in this release. But new orgs, scratch orgs and sandbox orgs have LWS for LWC & Aura components enabled by default, beginning from Spring'23 release itself.

How to enable ?
Setup → session Settings → "Use Lightning Web Security for Lightning web components and Aura components"
