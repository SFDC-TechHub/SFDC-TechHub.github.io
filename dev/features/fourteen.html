Apex: Access Labels in Apex Dynamically
Contributed by : Subhojeet Ghosh

From Summer '23, you can use the System.Label.get(namespace, label, language) method to get a custom label, optionally specifying a language.

Important Points :

  • Previously, retrieving labels required compile-time knowledge of the label API name.
  • This feature now allows dynamic resolution of label names at run time.
  • With this feature, if translation exists for a requested language, it will return the translated value overriding the user's current language.
  • Retrieve the label for a default language setting or for a language and namespace by using 
    System.Label.get(namespace, label, language)
    .
  • You can also check if translation exists for a label and language in a namespace by using 
    Label.translationExists(namespace, label, language)
    .
  • You cannot access labels that are protected in a different namespace.
