Apex: Manage DML Exceptions in User-Mode Database Operations
Contributed by : Subhojeet Ghosh

User mode means running apex code by respecting user's permissions (CRUD & FLS) and sharing of records. Previously user-mode DML operations generated a SecurityException. But now from Summer '23 release, they will generate the correct DMLException.

Important Points :

  • This change is supported in API version 
    58.0
    and later only.
