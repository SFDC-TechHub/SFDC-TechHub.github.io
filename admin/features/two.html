Service: Run the Lightning Knowledge Migration Tool
Contributed by : Subhojeet Ghosh

The Classic Knowledge data model retirement phase is in progress. Salesforce will enforce this update on June 1, 2025. To prepare for this retirement and take advantage of updated features, the Lightning Knowledge Migration Tool can be run.

Benefits:

  • Get the latest Knowledge features when you migrate from Knowledge Classic to Lightning Knowledge, like Einstein Article Recommendations, Einstein Search for Knowledge, Flow support, and bots.
  • After running the Lightning Knowledge Migration Tool, you can still use Classic Knowledge in the Classic UI.

After the migration tool is run, Lightning Knowledge works the following ways in Salesforce Classic:

  • Because Lightning Knowledge uses 
    record types
    instead of 
    article types
    , you can no longer filter by article type.
  • You cannot set a default record type in Salesforce Classic.
  • Some actions, such as 
    Change Record Type
    , are not available from the knowledge article record page in Salesforce Classic.
  • Picklist values assigned to one record type in Knowledge are not visible on records in Salesforce Classic.
  • Refactoring your custom solutions on your Classic Knowledge implementation might be needed.
