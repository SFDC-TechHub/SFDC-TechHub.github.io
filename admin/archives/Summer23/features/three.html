Admin: Limit the Number of Inactive Picklist Values
Contributed by : Subhojeet Ghosh

With the Summer '23 release, Salesforce is enforcing the limit of inactive picklist values to 4,000 for new unrestricted custom picklists to improve org health and performance. This change applies to Lightning Experience and Salesforce Classic in all editions.

Any unrestricted custom picklists that were previously created that have more than 4,000 inactive values remain unchanged.
