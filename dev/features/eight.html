Lightning Components: Assign CSS Stylesheets to a Component
Contributed by : Subhojeet Ghosh

From Summer '23, now you can extend stylesheets from a superclass component to a subclass component. Previously, CSS custom properties and CSS ::part() were the only standard mechanisms for customizing components and you could not directly reference stylesheets from an extended base class.

Important Points:

  • The new 
    stylesheets
    static property on the 
    LightningElement
    constructor loads styles from CSS stylesheets into both light DOM and shadow DOM components.
  • The 
    stylesheets
    property accepts a static array of stylesheets, and its default value is an empty array.
  • Stylesheets are injected into a component in the order that they're listed in the 
    stylesheets
    array.
