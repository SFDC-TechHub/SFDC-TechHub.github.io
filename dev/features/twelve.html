Apex: Use the Iterable Interface with Set Type
Contributed by : Subhojeet Ghosh

From Summer '23, Set class now implements the Iterable interface, so that you can directly iterate over sets. Lists were already iterable earlier. Now Sets are iterable too.

Important Points :

  • This change is supported in API version 
    58.0
    and later only.
