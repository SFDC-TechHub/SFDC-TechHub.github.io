Lightning Components: Create Scoped Slots in Light DOM
Contributed by : Subhojeet Ghosh

Light DOM LWC now supports Scoped Slots.

Use Case:

  • Scoped slots are used to access data from a child component and then render it in slotted content inside of a parent component.
  • Any style on the parent applies to the scoped slots because the slotted content belong to the parent component.

Considerations:

  • Child component must use light DOM. Scoped slots in shadow DOM arenâ€™t supported.
  • The parent component can be a light DOM or shadow DOM component.
  • To introduce scoped slots, add the directives 
    lwc:slot-data
    in parent component & 
    lwc:slot-bind
    in the child component.
